# Phone

This Phone app is a simplified version of [Fossify Phone](https://github.com/FossifyOrg/Phone)\
The goal of this app is to provide a simple, yet modern alternative to AOSP Dialer on ArfoxOS\
All credits goes to FossifyOrg
